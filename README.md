# Malcolm Live Dashboard

Real-time status dashboard for Malcolm (Windows PC agent).

## Live URL
**https://malcolm-status-dashboard.vercel.app**

## Features

- 🟢 **Live Status Indicator** - Green when active, yellow when idle, red when offline
- 📱 **Mobile Friendly** - Works perfectly on phones
- ⚡ **Auto-refresh** - Updates every 10 seconds
- 🌙 **Dark Theme** - Easy on the eyes

## Status Meanings

| Status | Color | Meaning |
|--------|-------|---------|
| ONLINE | 🟢 Green | Active in last 2 minutes |
| IDLE | 🟡 Yellow | Last seen 2-10 minutes ago |
| OFFLINE | 🔴 Red | No activity for 10+ minutes |

## Local Development

```bash
cd dashboard
python -m http.server 8080
# Open http://localhost:8080
```

## Deployment

This dashboard auto-deploys to Vercel on every push to the `main` branch.

### Manual Deploy
```powershell
# Sync status and deploy
.\scripts\dashboard-sync.ps1
```

## File Structure

```
dashboard/

```
